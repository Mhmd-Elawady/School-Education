# 🎓 School Education Website

A modern and responsive **school website** built with **React** and **Vite**.  
The project showcases various sections to highlight school details, activities, and student resources.

---

## 🚀 Features
- 🏫 **Home Page** with hero section and school details.  
- 📚 **About Section** describing school vision and mission.  
- 🎨 **Activities & Gallery** showcasing student life.  
- 📝 **Contact Form** for parents and students.  
- 📱 Fully **Responsive Design** for all devices.  
- ✨ Smooth **animations & transitions** for better UI/UX.  

---

## 🛠️ Tech Stack
- ⚛️ **React** – UI building  
- ⚡ **Vite** – development & bundling  
- 🎨 **Bootstrap** – responsive styling & layout  
- 🎯 **Formik** – form handling & validation  
- ✅ **Yup** – schema validation with Formik  
- 🎬 **Framer Motion** – animations & transitions  
- 🌐 **React Router DOM** – navigation & routing
